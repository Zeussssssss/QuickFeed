    Brief Overview

    This website application is built towards an audience of students and professors/teachers. The application is designed to allow students to passively send constructive criticism/feedback to a lecturer, meaning there is minimal effort needed by students to use this application.

    The goal of this application is for users to enter keystrokes/click a button to produce an automated message with a timestamp to be sent to the database. For example, if a user wanted to say “[LECTURER] is confusing at [TIMESTAMP] by [USER]” or “[LECTURER] moved too fast at [TIMESTAMP] by [USER]” they would simply hit a key/click a button to send a message that auto-populates that field based off set config to the database. At the end of a lecture period all models of messages in the database will be compiled/analyzed for a view by the professor through an email.

    How to use this web app?

    Even though we did our best to make this website user friendly, there can be stuff that still might be a little tricky to understand. Therefore, we have made this brief guide to help you out whenever and wherever you get stuck!

    Login/Signup Steps(for both students and instructors)

    The process for login in to an accouny(or creating a new account), is pretty simple and is quite similar to the login sections of other websites. The only thing that the user should keep in mind is that they should not forget to check the option to use QuickFeed as an Instructor in both Signup and login pages if they wish to use the added functionality that an instructor account would get. Also, instructors will not be able to login unless they select the login as instructor option.

    The links to the signup and the login pages are present on the main home page(in both the navigation bar and the main body of the page), and also both the pages have links to each other.

    Some Instructor Specific Tips/Instructions

    1. When you login as an Instructor, then you are taken to a page called the "Instructor Homepage". From this homepage you can do tasks such as adding course, starting a class etc. Whenever you want to exit your Instructor homepage, you can easily signout by clicking on the Signout button at the top right corner.
    2. If you want to add a course, then click on its button in the instructor homepage, then it would take you to a new page where you would be required to enter a few details about the course. After you do that, click on the Create course button which would create the course for you.You would then be able to see a confirmation message along with a link that you can share with your students so that they can add themselves to your course.
    3. If you want to see the feedback statistics for a particular course, then click on the that course's tile in the course section of the Instructor homepage, and then you would be taken to a new page which contains the all of the available feedback for the selected course
    4. Also, on that page you would be able to see an option to copy the link to your clipboard, which you can do if you misplace the link to the class that was given to you when it was created. Most importantly, there is a button START/STOP the class, which is pretty important to use this app. Please note: You can not receive feedback for a particular session until you start a class!After a session ends, stop the class and you would get the feedback for that particular session, along with the the updated stats for the entire semester.

    Some Student Specific Tips/Instructions

    1. When you login as a Student, then you are taken to a page called the "Student Homepage". From this homepage you can do tasks such as adding course, giving feedback for a class etc. Whenever you want to exit your Student homepage, you can easily signout by clicking on the Signout button at the top right corner.
    2. If you want to add a course, just enter the link that was given to you by your instructor in the respective section of the homepage. If the link is correct, then you would be enrolled in the course, else you would see an error message. If you feel that your link is correct and yet you are seeing an error, then check for any blank spaces else just contact your instructor. Remember, links are case sensitive
    3. To give feedback for an active course, just click on its tile in courses section and you would be taken to a new page. If the course is live, then you would see multiple buttons on which you can click to give your feedback.

    If you ever feel stuck, then just click on the QuickFeed logo and you would be taken to your respective homepage!
